To create a new countainer In Bash (cmd):
docker run --name dtc-postgres ^
  -e POSTGRES_USER=dtc_user ^
  -e POSTGRES_PASSWORD=dtc_pass ^
  -e POSTGRES_DB=dtc ^
  -p 5432:5432 ^
  -d postgres:16

To access the countainer:
docker exec -it dtc-postgres psql -U dtc_user -d dtc

To Import Data:
DROP TABLE IF EXISTS employees;
DROP TABLE IF EXISTS schools;

To cleanup database:
TRUNCATE TABLE employees, schools RESTART IDENTITY;

To Create tables:
CREATE TABLE schools (
    school_id SERIAL PRIMARY KEY,
    governorate TEXT,
    wilaya TEXT,
    school_code TEXT UNIQUE,
    school_name TEXT
);
CREATE TABLE employees (
    emp_id SERIAL PRIMARY KEY,
    school_id INTEGER,
    employee_name TEXT,
    nationality TEXT,
    contract_type TEXT,
    job_title TEXT,
    subject TEXT,
    service_years INTEGER,
    hire_date DATE,
    FOREIGN KEY (school_id) REFERENCES schools(school_id)
);
FOREIGN KEY (school_id)

To add users:

To find all tables:
\dt
To find name of columns of a table(users for example):
SELECT column_name
FROM information_schema.columns
WHERE table_name = 'users';
To find all records of a table:
SELECT * FROM users;
To find details of columns of a table:
\d users
To know each user whom admin or supervisor:
SELECT u.username, s.username AS supervisor
FROM users u
LEFT JOIN users s ON u.supervisor_id = s.id;

To rename column name:
ALTER TABLE files
RENAME COLUMN uploaded_at TO uploaded_date;

To check current users:
SELECT id, username, full_name, role, supervisor_id
FROM users
ORDER BY id;

To add table of tracking users:
CREATE TABLE IF NOT EXISTS activity_logs (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id) ON DELETE CASCADE,
    action_type VARCHAR(100),
    target_file VARCHAR(255),
    log_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

To track recording data:
SELECT * FROM activity_logs;
OR:
SELECT
    a.id,
    u.username,
    a.action_type,
    a.target_file,
    a.log_time
FROM activity_logs a
JOIN users u ON a.user_id = u.id
ORDER BY a.log_time DESC;

To run python files on Mac Terminal:

2- python3 -m venv venv
source venv/bin/activate
3- pip install fastapi uvicorn psycopg2-binary requests pandas openpyxl python-multipart
4- uvicorn api_server:app --reload
5- Uvicorn running on http://127.0.0.1:8000
To host:
cd ~/Desktop/DTC
source venv/bin/activate
uvicorn api_server:app --reload --host 127.0.0.1 --port 8000
